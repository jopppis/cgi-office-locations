# cgi-office-locations
CGI office location finder

Geocoding via Nominatim (OpenStreetMap) https://www.openstreetmap.org/copyright.

Scrape CGI office locations from https://www.cgi.com/ using the page layout at 2025-10-10.

## Usage

Running the tool requires `uv`. [Install it](https://docs.astral.sh/uv/getting-started/installation/) and then run:

```bash
./cgi-office-locations.sh
```
